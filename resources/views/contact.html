Pertanyaan Umum

Bagaimana cara memulai menggunakan KlikAja?

Anda bisa langsung menggunakan KlikAja tanpa registrasi! Cukup paste URL panjang Anda di homepage dan klik "Perpendek". Untuk fitur advanced seperti analytics dan management, silakan daftar gratis.

Apakah KlikAja benar-benar gratis?

Ya! KlikAja adalah 100% open source dan gratis. Hosted version kami gratis dengan limit 100 link/bulan. Untuk unlimited, Anda bisa self-host di server sendiri secara gratis.

Bagaimana cara self-host KlikAja?

Ikuti panduan lengkap di GitHub README. Anda membutuhkan Node.js 18+ dan SQLite. Proses instalasi hanya memakan waktu 5-10 menit.

Bagaimana cara melaporkan bug?

Silakan buat issue di GitHub Issues. Sertakan detail lengkap tentang bug, langkah reproduksi, dan screenshot jika memungkinkan.

Apakah tersedia API untuk developer?

API sedang dalam development dan akan tersedia di Q1 2026. Follow GitHub repository untuk update terbaru.

Bagaimana cara berkontribusi?

Kami welcome semua kontribusi! Fork repository, buat branch baru, commit changes Anda, dan submit Pull Request. Baca Contributing Guide untuk detail lengkap.
