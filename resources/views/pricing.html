Pertanyaan Umum

Apakah KlikAja benar-benar gratis?

Ya! KlikAja adalah open source dengan lisensi MIT. Anda bisa menggunakan, memodifikasi, dan deploy tanpa biaya selamanya. Hosted version kami juga gratis dengan limit yang wajar.

Apa bedanya self-hosted vs hosted version?

Self-hosted: Anda deploy di server sendiri, unlimited semua, full control.
Hosted version: Kami yang maintain server, Anda tinggal pakai, ada limit bulanan.

Kapan fitur Pro akan tersedia?

Kami sedang mengembangkan fitur-fitur premium seperti custom domains, team collaboration, dan advanced analytics. Target launch Q1 2026. Follow GitHub untuk update!

Bagaimana cara support project ini?

Star repository di GitHub, contribute code, report bugs, share ke komunitas, atau donate via Ko-fi. Setiap dukungan sangat berarti!
